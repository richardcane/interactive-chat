# Interactive Chat ![Plugin icon](https://raw.githubusercontent.com/richardcane/interactive-chat/master/icon.png)

### This plugin is currently in development. If you find any bugs not listed under Issues, please report them.

This plugin allows players to send and recieve messages with interactive keywords. As of now, these interactive keywords open the wiki page for the given keyword, or if the page doesn't exist, it'll open a wiki search. It does not currently work with the Fixed game client layout.
